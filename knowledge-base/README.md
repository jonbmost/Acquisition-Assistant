# Knowledge Base Documents

This directory contains persistent documents that the AI assistant can reference when answering questions and drafting solutions.

## Document Categories

### Federal Acquisition Regulations (FAR)
- Add FAR guidance documents here

### Templates
- Statement of Objectives (SOO)
- Performance Work Statement (PWS)
- Quality Assurance Surveillance Plan (QASP)
- Security and Privacy Plans (SSP)
- Evaluation criteria templates

### Agency Playbooks
- Leading Agile Acquisitions
- TechFAR Hub guidance
- Agency-specific handbooks

### Best Practices
- Modular contracting guides
- Agile acquisition case studies
- Market research templates

## Adding Documents

1. Place files in this directory or create subdirectories
2. Supported formats: .txt, .md, .pdf, .docx
3. Commit and push to make them available to all users

## Loading into the App

The app will automatically load these files at startup from the repository.
